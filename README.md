## What is this?

**githubmaxxing** scores your GitHub profile across 5 categories and gives you a brutally honest, ranked action plan. No login. No email. 100% public API.

```
┌─────────────────────────────────────────────────────────┐
│  @dhivyeshp                              B tier          │
│                                          64 / 100        │
│  good bones · fix README                                 │
│                                                          │
│  Profile   ████████░░░░  55                              │
│  Repos     ██████████░░  70                              │
│  READMEs   ████████░░░░  50                              │
│  Commits   ███████████░  75                              │
│  Social    ████░░░░░░░░  25                              │
└─────────────────────────────────────────────────────────┘
```

## Scoring

| Category | Weight | What it checks |
|---|---|---|
| Profile Completeness | 55 | Avatar, bio, location, website, profile README |
| Repository Quality | 70 | Original repos, descriptions, active maintenance |
| README Quality | 50 | Documentation quality across top repos |
| Commit Consistency | 75 | How recently and consistently you push code |
| Social Proof | 25 | Followers and star count signal |

Grades:

```
A tier  80–100   ██████████  you're hireable, keep it up
B tier  60–79    ████████░░  good bones, quick wins available
C tier  0–59     ████░░░░░░  needs work, action plan incoming
```

## Stack

```
app/
├── page.jsx          # Homepage — hero, input, social proof cards
├── results/
│   └── page.jsx      # Results — score breakdown + action plan
├── api/
│   ├── analyze/      # GitHub data fetching + scoring
│   └── insights/     # Claude AI action plan generation
│
components/
├── ScoreCard.jsx     # Per-category score card
├── ActionPlan.jsx    # Prioritized fix list
└── ShareCard.jsx     # Twitter share strip
│
lib/
├── github.js         # GitHub API client
├── scorer.js         # Scoring logic (5 categories)
└── prompts.js        # Claude system prompt (GitHub Maxx persona)
```

- **Framework:** Next.js 15 (App Router)
- **Styling:** Inline styles + CSS custom properties
- **Background:** `@paper-design/shaders-react` — animated mesh gradient
- **AI:** Claude API for action plan generation
- **Deploy:** Vercel

## Running locally

```bash
# 1. Clone
git clone https://github.com/Dhivyeshp/githubmaxxing.git
cd githubmaxxing

# 2. Install
npm install

# 3. Set env vars
cp .env.example .env.local
# Add your GITHUB_TOKEN and ANTHROPIC_API_KEY

# 4. Run
npm run dev
# → http://localhost:3000
```

### Environment variables

| Variable | Required | Description |
|---|---|---|
| `GITHUB_TOKEN` | Yes | GitHub fine-grained PAT (public repos, read-only) |
| `ANTHROPIC_API_KEY` | Yes | Claude API key for action plan generation |
